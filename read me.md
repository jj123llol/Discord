# install guide

## PC
install python
pip install discord.py-self or discord.py,

pip install PyNaCl

run sb.py


## Mobile (Android)
install pydroid 3

tap the 3 line button, and goto pip.

goto the install tab and enter
discord.py-self, and then uwuipy.
run the code from sb.py in the app.
